# The fun and (mostly)free way to learn programming

[< Back Home](/)

Use the resources that you have fun using and most relate with
Take at your own pace

## [Boot.dev](www.boot.dev)

![Boots](/images/boots.webp)

- Gamified curated courses for backend development
- Very fun game mechanics, you get chests, shields, potions and moree.
- A very helpful community
- Courses that start from very fine steps to
- Emphasises on why you would build something, instead of just How
- Good problem scenarios
- Lots of fun projects to go through and expand on
- You get boots as your assistant, never answers directly but leads you to the right path

### Quick Start

- Create a free account at Boot.dev
- Start with "Learn Python" as your first course
- Complete coding challenges directly in the browser
- Track your progress through the gamified achievement system
- Join the Discord community for help and motivation
- If you like and want to buy the paid subscription you can find a 25% off coupon pretty easily

## [The Odin Project](https://www.theodinproject.com/)

![Odin](/images/odin.png)

- Completely free open source resource
- Focused on frontend and web app development
- Very thorough courses
- Project-based curriculum that builds real portfolio pieces
- Strong focus on problem-solving and self-sufficiency
- Active Discord community with peer support

### Quick Start

- Sign up for a free account to track progress
- Begin with the "Foundations" course (HTML, CSS, JavaScript basics)
- Follow the curriculum sequentially, each lesson builds on the previous
- Complete all projects, they form your portfolio
- Join the Discord for help when stuck

## [FreeCodeCamp](https://www.freecodecamp.org/)

![FreeCode](/images/freecode.png)

- More balanced fullstack courses
- Offers certificates for completion
- Provides free and comprehensive tutorials on web development and data science
- Interview and job prep resources
- Amazing community
- Hands-on coding challenges directly in the browser

### Quick Start

- Create a free account at FreeCodeCamp
- Start with "Responsive Web Design" certification
- Code directly in the browser with instant feedback
- Complete all challenges to earn your first certificate
- Explore the forum for help and community support

## [Coursera](https://www.coursera.org/)

![Coursera](/images/coursera.png)

- Lots of courses from very reputable institutions
- Varying level of difficulties from beginner to expert
- Multi level courses that cover all aspects of that topic
- Some of them can be found for free
- Includes most fields possible
- Certificates that can be helpful for your resume
- College like experience with quizzes and exams that test your progress

### Quick Start

- Sign up for a Coursera account
- Browse courses by topic or search for specific skills
- Audit courses for free or pay for certificates
- Recommended: "Python for Everybody" or "Google IT Automation with Python"
- Follow weekly structure with videos, quizzes, and assignments
- Apply for financial aid if you need certificate access

## OpenCourseware

![OCW](/images/ocw.jpg)

### [MIT OCW](https://ocw.mit.edu/search/?d=Electrical%20Engineering%20and%20Computer%20Science)

- Free lecture notes, exams, and videos from actual MIT courses
- Covers CS fundamentals like 6.006 Algorithms and 6.824 Distributed Systems
- Self-paced learning with no registration required

**Quick Start:**

- Visit ocw.mit.edu, no account needed
- Start with 6.0001 "Introduction to Computer Science and Programming in Python"
- Download lecture notes, watch videos, attempt problem sets
- Use the course materials at your own pace

### [CMU OLI](https://oli.cmu.edu/product-category/computer-science-programming/)

- Interactive online courses with immediate feedback
- Strong computer science theory foundations
- Built-in assessments track your understanding

**Quick Start:**

- Go to oli.cmu.edu and create a free account
- Browse available CS courses
- Start with interactive modules and receive immediate feedback
- Use built-in assessments to gauge understanding
